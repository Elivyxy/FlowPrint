.. _Cluster:

Cluster
=======
After performing feature extraction, FlowPrint clusters all :ref:`Flow`'s into :py:obj:`cluster.NetworkDestination` according equal (destination IP, destination port)-tuple or TLS certificates.

.. autoclass:: cluster.Cluster

    .. automethod:: __init__

Generating clusters
^^^^^^^^^^^^^^^^^^^
We can create clusters from :ref:`Flow`'s by fitting the Cluster object :py:meth:`cluster.Cluster.fit` method.
After fitting the cluster, we can use the :py:meth:`cluster.Cluster.predict` method to get all cluster labels as numbers.
The :py:meth:`cluster.Cluster.fit_predict` method combines both methods into a single action.

.. automethod:: cluster.Cluster.fit

.. automethod:: cluster.Cluster.predict

.. automethod:: cluster.Cluster.fit_predict

Cluster views
^^^^^^^^^^^^^
We extract the different :py:obj:`cluster.NetworkDestination`'s generated by the cluster either as a set or as a dictionary of `identifier -> NetworkDestination`.
